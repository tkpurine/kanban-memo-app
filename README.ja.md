# カンバンメモアプリ

日々の活動をサポートするカンバン形式のタスク管理アプリです。セッション単位で作業を記録し、JSONファイルにデータを保存します。

## 機能

- **カンバンボード** — Todo / 作業中 / 待ち / 完了 の4列構成
- **セッション管理** — セッションの開始・終了、未完了タスクの自動引き継ぎ
- **ドラッグ&ドロップ** — タスクの列間移動、タグのタスクへの割り当て
- **タグシステム** — タグの作成・付与によるタスク整理
- **ローカルJSON保存** — 指定フォルダにJSONファイルとしてデータを保存

## 技術スタック

| レイヤー | 技術 |
|----------|------|
| フロントエンド | HTML / CSS / JavaScript |
| バックエンド | Node.js（Express） |
| データ | JSONファイル（ローカルフォルダ） |

## 使い方

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開き、データ保存先のフォルダパスを指定してください。
